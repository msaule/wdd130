Portal Page Plan

WDD 130 progress check 100 design document

Student: Markuss Saule

Site Purpose

This portal page will be the main home page for all class work in WDD 130. It will act as a simple portfolio where visitors can quickly find my rafting site, ICE challenge work, and an about me page. I want it to feel clean, organized, and personal instead of looking like a generic template.

Planned Sections

  • Header: logo, site title, and navigation for Home, WWR, ICE, and About Me.
  • Hero: short welcome message with a profile image and a sentence about the class.
  • Main project area: feature cards linking to current coursework like WWR and ICE pages.
  • Checklist block: an unordered list with current goals or weekly tasks.
  • Course table: a small table showing project names, status, and links.
  • Contact form: a simple form for practice with labels, inputs, and a button.
  • Footer: copyright line and quick navigation links.

Wireframe

The layout uses a clear top-to-bottom flow. The first section introduces the page, the middle area highlights course projects, and the lower part contains the table and form so the home page includes all required elements while still feeling organized.

Color Palette

I changed the palette to use navy, rust, and warm paper colors so the portal feels more grounded and personal. It still looks clean, but it has more contrast and a little more personality.

Name Hex Use
Ink #22303C Header, footer, main headings
Slate #66727F Secondary accents and borders
Clay #B56F50 Callouts, buttons, and small highlights
Paper #F4EFE7 Page background
Graphite #2B2B2B Main text

Typography

Headings: Bree Serif, Georgia, serif

Body: Source Sans 3, Verdana, sans-serif

I want the page to feel current and easy to scan, so the headings use a friendly display font and the body copy uses a clean sans-serif. This keeps project descriptions readable while giving the portal a little more personality.

Logo Idea

The logo is now a simple MS monogram for Markuss Saule. I wanted it to feel more like a real portfolio mark that could sit in the corner of a website or resume instead of a class icon.
