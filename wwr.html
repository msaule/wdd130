Main Project

White Water Rafting Website

The rafting site has been the biggest project in the course so far. It includes planning, HTML structure, CSS styling, layout, responsive changes, and multiple checkpoints.

What I Learned

This project pushed me to work through the whole process instead of just writing one page. I had to plan first, then build the structure, then style it, and finally improve layout and spacing.

  • Using semantic HTML more carefully
  • Working with color and typography choices
  • Building layouts with CSS grid and flexbox
  • Making the site adapt better to smaller screens
